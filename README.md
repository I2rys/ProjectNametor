## Installation
Github:

    git clone https://github.com/I2rys/ProjectNametor
    
NPM Packages:
```
npm i project-name-generator && npm i express && npm i j2url
```

Test NPM Packages:
```
npm i got
```

## Usage
Default start:
```
node index.js
```

NPM start:
```
npm start
```

## Dev Usage
NPM start dev:
```
npm run start:dev
```

NPM test:
```
npm test
```
